До версии 1.21.11 код был обфусцирован, поэтому требовалась настройка маппингов, но целевая версия сервера - >=1.21.11, так что эта документация поможет только с этими версиями.

## Требования

Ниже приведены команды для 1.21.11 и для снапшота 26.1, но вы можете использовать любую версию.
Используйте лаунчер Minecraft (также Prism и др.), создайте экземпляр с нужной версией, запустите его - это загрузит jar-файл. Его расположение зависит от лаунчера и ОС. Получив jar-файл, выполните один из вариантов ниже.

## Существует несколько способов

- Использование `update-minecraft-src.sh`, нашего скрипта декомпиляции из репозитория (он использует `gitcraft` в качестве третьего способа за кулисами).

- Онлайн декомпилированная версия, где можно отправлять ссылки на файлы другим людям. Сайт: [mcsrc.dev](https://mcsrc.dev) можно использовать как [https://mcsrc.dev/#1/1.21.11_unobfuscated](https://mcsrc.dev/#1/1.21.11_unobfuscated).

- Просмотр исходного кода Minecraft через клонирование [gitcraft](https://github.com/WinPlay02/GitCraft) и запуск
  `./gradlew run --args="--only-stable --min-version=1.21.11 --only-unobfuscated"` - это создаст
  `minecraft-repo-mojmap-unobfuscated-min-1.21.11-stable/minecraft` со всем исходным кодом.

- Скачать [vineflower.jar](https://github.com/Vineflower/vineflower/releases) и выполнить
  `java -jar vineflower-1.11.2.jar ./minecraft-26.1-snapshot-3-client.jar --folder minecraft_26.1`. Это создаст папку `minecraft_26.1`.
