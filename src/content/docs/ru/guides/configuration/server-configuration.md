SteelMC настраивается через JSON5-файл конфигурации, расположенный в `config/steel_config.json5`. На этой странице описаны все доступные параметры (вроде как xd).

## Основные настройки

| Параметр | Тип | По умолчанию | Описание |
|--------|------|---------|-------------|
| `server_port` | u16 | `25565` | Порт, который слушает сервер (1-65535) |
| `seed` | String | `""` | Сид генерации мира (пустая строка = случайный) |
| `max_players` | u32 | `20` | Максимальное количество одновременных игроков |
| `view_distance` | u8 | `10` | Максимальная дальность обзора в чанках (1-32) |
| `simulation_distance` | u8 | `10` | Максимальная дистанция симуляции в чанках (1-32) |
| `motd` | String | `"A Steel Server"` | Сообщение, отображаемое в списке серверов* |

*motd - Message of the day, сообщение дня, ну или же просто описание сервера.

## Настройки безопасности

| Параметр | Тип | По умолчанию | Описание |
|--------|------|---------|-------------|
| `online_mode` | bool | `true` | Использовать аутентификацию Mojang для проверки игроков |
| `encryption` | bool | `true` | Включить шифрование для соединений игроков |
| `enforce_secure_chat` | bool | `false` | Принудительно использовать безопасный чат (требует `online_mode` и `encryption`) |

:::caution
Отключение `online_mode` позволяет подключаться пиратским клиентам. Отключайте только если понимаете, что делаете, или для частных сетей и разработки.
:::

## Настройки фавиконки

| Параметр | Тип | По умолчанию | Описание |
|--------|------|---------|-------------|
| `use_favicon` | bool | `true` | Использовать ли иконку для сервера |
| `favicon` | String | `"config/favicon.png"` | Путь к файлу иконки (64x64 PNG) |

## Настройки сжатия

Сжатие сетевого трафика снижает использование пропускной способности ценой нагрузки на процессор.

| Параметр | Тип | По умолчанию | Допустимый диапазон | Описание |
|--------|------|---------|-------------|-------------|
| `compression.threshold` | u32 | `256` | ≥256 | Порог размера пакета для сжатия |
| `compression.level` | u8 | `4` | 1-9 | Уровень сжатия (1=быстро, 9=максимально) |

## Ссылки сервера

Ссылки сервера отображаются в меню многопользовательской игры.

| Параметр | Тип | По умолчанию | Описание |
|--------|------|---------|-------------|
| `server_links.enable` | bool | `true` | Включить функцию ссылок сервера |
| `server_links.links` | Array | См. ниже | Список отображаемых ссылок |

Подробнее см. [Руководство по ссылкам сервера](../server-links).

## Пример конфигурации

```json5
// /config/steel_config.json5
{
  server_port: 25565,
  seed: "my_awesome_seed",
  max_players: 50,
  view_distance: 12,
  simulation_distance: 10,
  online_mode: true,
  encryption: true,
  motd: "Welcome to my Steel server!",
  use_favicon: true,
  favicon: "config/favicon.png",
  enforce_secure_chat: false,
  compression: {
    threshold: 256,
    level: 4
  },
  server_links: {
    enable: true,
    links: [
      {
        label: { text: "Discord" },
        url: "https://discord.gg/example"
      }
    ]
  }
}
```

## Правила валидации

Сервер проверяет конфигурацию при запуске:

- `view_distance` должен быть от 1 до 64
- `simulation_distance` должен быть от 1 до 32
- `compression.threshold` должен быть не менее 256
- `compression.level` должен быть от 1 до 9
- Если `enforce_secure_chat` включён, то `online_mode` и `encryption` должны быть включены
